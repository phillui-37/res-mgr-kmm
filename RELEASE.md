# Release Notes

## Version 1.0.0

This is the first stable release of the Resource Manager system, **completely generated by AI**.

### Highlights
- Unified resource management for various digital assets.
- Cross-platform desktop client (Windows, macOS, Linux).
- Lightweight self-hosted server.
- Efficient batch import for large libraries.

### Installation

#### Server
Run the startup script in `bin/server` after building.

#### Client
Use the native distribution created in the build folder.
